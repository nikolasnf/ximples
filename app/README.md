# 🚀 Ximples - Seu Marketing Feito por Você. Só Que Sem Você.

Bem-vindo ao **Ximples**, um operador digital de marketing via chat que executa suas estratégias de marketing automaticamente.

## 📋 Conceito

Ximples é um chatbot inteligente que interpreta suas intenções de marketing e executa tudo automaticamente:

- **Landing Pages** - Cria pages de vendas otimizadas
- **Campanhas de Email** - Gera sequências de emails persuasivas  
- **Mensagens WhatsApp** - Desenvolve campanhas automatizadas por WhatsApp
- **Funis de Vendas** - Monta funis completos de conversão
- **CRM & Leads** - Organiza seus leads em um pipeline

## 🎯 Como Funciona

1. **Digite sua intenção** - "Quero vender um ebook de produtividade"
2. **Ximples responde** - Mostra o que será criado
3. **Marcos de progresso** - Acompanha cada etapa em tempo real
4. **Ativos prontos** - Acessa tudo que foi criado na biblioteca

### Exemplo de Uso

```
Usuário: "Quero vender meu curso de inglês com landing, email e WhatsApp"

Ximples: "Perfeito. Estou criando tudo para você agora.
✓ Landing criada
✓ Funil criado  
✓ Campanha ativa
⏳ Leads chegando
```

## 🎨 Interface

### Componentes Principais

- **Chat Central** - Interação com o Ximples
- **Sidebar Esquerdo** - Histórico de conversas
- **Painel de Marcos** - Progresso visual das tarefas
- **Biblioteca de Ativos** - Tudo que já foi criado

## 🛠️ Stack Técnico

- **Frontend**: React + Next.js 16
- **UI**: shadcn/ui + Tailwind CSS
- **Ícones**: Lucide React
- **Responsivo**: Mobile-first design
